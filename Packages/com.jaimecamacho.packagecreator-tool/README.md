# Package Creator Tool

Unity package `com.jaimecamacho.packagecreator-tool`.

## Installation

Add the package by name in the Unity Package Manager or from a scoped registry that exposes `com.jaimecamacho.packagecreator-tool`.

## Contents

- `Runtime/` runtime code.
- `Editor/` editor-only tools.
- `Documentation~/` package documentation.
- `Samples~/` optional importable samples.

## Development

Update `package.json`, `CHANGELOG.md` and `README.md` before creating a release.
